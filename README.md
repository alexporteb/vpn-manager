# Интерактивный VPN Менеджер (IKEv2, L2TP, PPTP)

Этот инструмент превращает ваш Ubuntu/Debian сервер в полноценный VPN-сервер с удобным **интерактивным консольным меню**.

Вам больше не нужно запоминать сложные команды: устанавливайте сервер, управляйте пользователями и просматривайте доступы через простое текстовое меню.

## Установка (Одной командой)

Подключитесь к вашему VPS по SSH и выполните команду от имени `root` (или с `sudo`):

```bash
curl -sL https://raw.githubusercontent.com/alexporteb/vpn/main/install.sh | sudo bash
```

Этот скрипт установит глобальную утилиту `vpn` и сразу же запустит главное меню.

## Использование меню

В любой момент, когда вы зайдете на свой сервер, просто напишите в консоли:

```bash
sudo vpn
```

Перед вами появится интерактивное меню:
1. **Установить VPN сервер** — автоматически скачает пакеты, сгенерирует ключи и поднимет все протоколы (IKEv2, L2TP, PPTP).
2. **Управление пользователями** — открывает подменю, где можно добавлять, удалять или смотреть список пользователей для всех протоколов сразу.
3. **Данные сервера** — показывает ваш IP и секретный ключ (PSK) для подключения.
4. **Удалить VPN сервер** — безопасно сносит все пакеты и очищает настройки фаервола.
5. **Выход** — закрывает меню.

---

## Как подключаться с устройств
- **IKEv2**: Потребуется имя пользователя и пароль. Самый быстрый и безопасный протокол. Идеально для iOS/macOS (выбирайте тип IKEv2).
- **IKEv1 (Cisco IPsec)**: Потребуется IP сервера, Общий ключ (PSK), имя пользователя и пароль. (Выбирайте тип Cisco IPsec).
- **L2TP/IPsec**: Потребуется IP-адрес сервера, Общий ключ (PSK) (можно посмотреть в пункте меню "Данные сервера"), имя пользователя и пароль.
- **PPTP**: Потребуется только IP-адрес сервера, имя пользователя и пароль. (Менее безопасен, но поддерживается старыми роутерами и ОС).
