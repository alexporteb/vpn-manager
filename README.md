# Автоматическая установка VPN сервера (IKEv2, L2TP, PPTP)

Этот скрипт позволяет в одну команду установить и настроить полноценный VPN-сервер на **Ubuntu / Debian** Linux. Поддерживает протоколы:
- **IKEv2** (самый современный, работает нативно на iOS, macOS, Windows).
- **L2TP/IPsec**
- **PPTP**

## Установка (Одной командой)

Подключитесь к вашему VPS по SSH и выполните следующую команду от имени `root` (или с `sudo`):

```bash
curl -sL https://raw.githubusercontent.com/alexporteb/vpn/main/vpn-server-install.sh | sudo bash
```

По окончании установки скрипт:
1. Выведет на экран **IP-адрес сервера**.
2. Сгенерирует надежный **Общий ключ (PSK)** для L2TP/IPsec.
3. Установит глобальную команду `vpn-manage-users` для простого добавления пользователей.

## Управление пользователями

Скрипт добавляет удобную утилиту `vpn-manage-users`, которая создает доступы сразу для всех протоколов.

### Добавить пользователя (или изменить пароль):
```bash
sudo vpn-manage-users add имя_пользователя пароль
```
*Пример: `sudo vpn-manage-users add alex mySecret123`*

### Удалить пользователя:
```bash
sudo vpn-manage-users del имя_пользователя
```

### Показать список пользователей:
```bash
sudo vpn-manage-users list
```

## Подключение с устройств
- **IKEv2**: Потребуется имя пользователя, пароль. (Для некоторых систем может потребоваться установка корневого сертификата `/etc/ipsec.d/cacerts/ca-cert.pem`).
- **L2TP/IPsec**: Потребуется IP-адрес сервера, Общий ключ (PSK), имя пользователя и пароль.
- **PPTP**: Потребуется IP-адрес сервера, имя пользователя и пароль.

## Удаление сервера (Одной командой)

Если вы хотите полностью удалить VPN сервер и все его компоненты с вашего VPS:

```bash
curl -sL https://raw.githubusercontent.com/alexporteb/vpn/main/vpn-server-remove.sh | sudo bash
```
